# EDA in Pandas - World Population

A beginner-friendly EDA project analyzing global population trends from 1970 to 2022 using Python.

## Dataset
`world_population.csv` — 234 countries with population figures across 8 decades, plus area, density, growth rate, and world population percentage.

## What's Covered
- Data inspection (`info`, `describe`, null checks, unique counts)
- Top 10 most populous countries in 2022
- Correlation heatmap across all numeric features
- Population trends by continent (1970–2022)
- Boxplot distribution of population metrics

## Requirements
```
pandas
matplotlib
seaborn
```

## Usage
```bash
pip install pandas matplotlib seaborn
jupyter notebook EDA_in_Pandas.ipynb
```
